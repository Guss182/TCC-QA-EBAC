# TCC QA EBAC – Projeto Final

## Objetivo
Consolidar práticas de qualidade de software com automações de UI, API, Mobile, Performance e CI, conforme as user stories fornecidas.

## Estrutura do repositório
- /docs → documentação (estratégia, critérios, casos de teste, evidências)
- /tests/ui → automação UI
- /tests/api → automação API
- /tests/mobile → automação Mobile
- /performance → scripts de performance (ex.: k6)
