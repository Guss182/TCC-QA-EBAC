# User Stories (US)

---

## US001 — Adicionar item ao carrinho

### Identificação
Projeto: EBAC-SHOP  
Versão afetada: 1.0  
Tipo: História  
Prioridade: Média  
Pontuação: 13  

---

### Descrição
Como cliente da EBAC-SHOP  
Quero adicionar produtos ao carrinho  
Para realizar a compra dos itens  

---

### Regras de Negócio

- Não é permitido inserir mais de 10 itens de um mesmo produto ao carrinho;
- O valor total do carrinho não pode ultrapassar R$ 990,00;
- Valores entre R$ 200,00 e R$ 600,00 ganham cupom de 10%;
- Valores acima de R$ 600,00 ganham cupom de 15%.

---

### Critérios Funcionais Esperados

- O sistema deve permitir adicionar produto ao carrinho;
- O sistema deve calcular corretamente o subtotal;
- O sistema deve aplicar automaticamente a regra de cupom conforme o valor total;
- O sistema deve impedir a adição de mais de 10 unidades do mesmo produto;
- O sistema deve impedir que o total ultrapasse R$ 990,00.

## US002 — Login na plataforma

### Identificação
Projeto: EBAC-SHOP  
Versão afetada: 1.0  
Tipo: História  
Prioridade: Média  
Pontuação: 8  

---

### Descrição
Como cliente da EBAC-SHOP  
Quero fazer login na plataforma  
Para visualizar meus pedidos  

---

### Regras de Negócio

- Somente usuários ativos podem fazer login;
- Deve exibir mensagem de erro caso o usuário informe login ou senha incorretos;
- Se o usuário errar a senha por 3 vezes, o login deve ser bloqueado por 15 minutos.

---

### Critérios Funcionais Esperados

- O sistema deve permitir login para usuários ativos com credenciais válidas;
- O sistema deve impedir login de usuários inativos;
- O sistema deve exibir mensagem de erro para login ou senha incorretos;
- O sistema deve bloquear o acesso após 3 tentativas inválidas de senha;
- O sistema deve impedir novo acesso durante o período de bloqueio.

## US003 — API de Cupons

### Identificação
Projeto: EBAC-SHOP  
Versão afetada: 1.0  
Tipo: História  
Prioridade: Média  
Pontuação: 13  

---

### Descrição
Como administrador da EBAC-SHOP  
Quero utilizar o serviço de cupons  
Para listar e cadastrar cupons no sistema  

---

### Autenticação da API

- Usuário: admin_ebac  
- Senha: @admin!&b@c!2022  
- Authorization: Basic YWRtaW5fZWJhYzpAYWRtaW4hJmJAYyEyMDIy  

---

### Regras de Negócio

#### GET
- Deve listar todos os cupons cadastrados;
- Deve permitir buscar cupom por ID.

#### POST
- Deve cadastrar cupons com os campos obrigatórios:
  - code (ex: "Ganhe10")
  - amount ("10.00")
  - discount_type ("fixed_product")
  - description ("Cupom de teste")
- O código do cupom não pode ser repetido;
- Os demais campos são opcionais.

---

### Critérios Funcionais Esperados

- A API deve permitir listar os cupons cadastrados;
- A API deve permitir consultar um cupom por ID;
- A API deve permitir cadastrar um cupom com os campos obrigatórios preenchidos;
- A API não deve permitir cadastro de cupom com nome repetido;
- A API deve retornar erro ao tentar cadastrar cupom sem os campos obrigatórios.

## US004 — Catálogo de Produtos

### Identificação
Projeto: EBAC-SHOP  
Versão afetada: 1.0  
Tipo: História  
Prioridade: Média  
Pontuação: 8  

---

### Descrição
Como cliente da EBAC-SHOP  
Quero visualizar o catálogo de produtos  
Para encontrar os itens que desejo comprar  

---

### Regras de Negócio

- O sistema deve permitir acessar os detalhes de um produto;

### Descrição

Quero acessar o painel Minha Conta  
Para visualizar e gerenciar minhas informações  

---

- Somente usuários autenticados podem acessar o painel Minha Conta;
